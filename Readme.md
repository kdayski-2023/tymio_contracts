# Tymio Contracts

Репозиторий смарт-контрактов и вспомогательной инфраструктуры для продукта **Tymio**: лимитные ордера, защита цены и реферальная логика на EVM-сетях с интеграцией **Uniswap V3**.

## Назначение

Проект собирает набор контрактов, которые:

- принимают депозиты в разрешённых токенах и ведут учёт балансов пользователей;
- создают и исполняют ордера с параметрами цены, срока действия и дополнительных выплат;
- для сценариев обмена используют маршрутизатор Uniswap V3 (`ISwapRouter`);
- предоставляют отдельный контракт для сценариев **защиты цены** (фиксация условий исполнения во времени);
- поддерживают **реферальные начисления** в стейблкоине (USDC).

Сборка и тесты — через **Hardhat** (Solidity `0.8.20`, оптимизатор включён).

## Основные контракты

| Контракт | Файл | Кратко |
|----------|------|--------|
| **PayerV3** | `contracts/PayerV3.sol` | Ордера на обмен `tokenIn` → `tokenOut`, балансы, исполнение через Uniswap, сроки (`endTimestamp`), роли владельцев и сервисного аккаунта. |
| **PayerV3Base** | `contracts/PayerV3Base.sol` | Базовая/общая логика для вариантов Payer (наследование). |
| **PriceProtectionV1** | `contracts/PriceProtectionV1.sol` | Защита цены: ордера по одному токену с `executionTimestamp`, лимиты по длительности и времени исполнения. |
| **ReferralV3** | `contracts/tools/ReferralV3.sol` | Реферальный кошелёк на USDC: балансы пользователей, вывод под контролем владельцев/сервиса. |

В `depricated_contracts/` лежат старые версии (например, PayerV1, ReferralV2) — для справки, не для новых интеграций.

## Вспомогательные библиотеки

- `contracts/lib/` — ошибки (`Errors.sol`, `ErrorsPriceProtection.sol`), `ArrayUtils.sol`.
- `contracts/interface/` — интерфейсы роутеров Uniswap (`ISwapRouter`, `ISwapRouter02`).
- `contracts/dev/` — тестовые ERC20, WETH9, mock-роутер для локальной разработки.

## Развёрнутые адреса

Актуальные адреса для разных сетей перечислены в `deployments.js` (Ethereum, Arbitrum, Base, тестовые Sepolia/Arbitrum Sepolia, Mumbai и т.д.). При интеграции сверяйтесь с этим файлом и с блокчейн-обозревателем.

## Скрипты

- **`scripts/deploy/`** — деплой PayerV3, PriceProtectionV1, ReferralV3 под конкретные сети.
- **`scripts/verify/`** — верификация исходников на обозревателях.
- **`scripts/expiration/`**, **`scripts/adminPanel/`** — проверки сроков, выплаты, балансы, служебные операции.
- **`scripts/decode/`** — разбор данных/логов контрактов.

## RPC-сервис

`rpc.js` поднимает HTTP-сервер (по умолчанию порт **5001**) с JSON-RPC эндпоинтами `/rpc` и `/tymio` (логика в `rpc/router.js`) для взаимодействия бэкенда с проверками и служебными методами.

## Требования и установка

- Node.js и npm
- Переменные окружения: в корне ожидается `.env` с `DEPLOYER_PRIVATE_KEY` и при необходимости ключами для Pinata/IPFS (см. `hardhat.config.js`).

```bash
npm install
npx hardhat compile
```

## Команды npm

| Команда | Описание |
|---------|----------|
| `npm run compile` | Компиляция контрактов |
| `npm test` | Запуск тестов Hardhat |
| `npm run cover` | Покрытие кода (solidity-coverage) |
| `npm run dev` | Запуск RPC и UI (если настроен каталог `ui`) |

Тесты лежат в `test/`; дополнительные/устаревшие сценарии — в `test_old/`, `old_tests/`.
